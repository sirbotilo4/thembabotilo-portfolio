SCSS - Themba Botilo Portfolio

The style.css file in assets/css/ is the working stylesheet.
This folder contains the Sass (.scss) source file if you prefer
to edit styles using Sass and compile to CSS.

To use Sass:
  1. Install Node.js from https://nodejs.org
  2. Install Sass globally: npm install -g sass
  3. Compile: sass assets/scss/style.scss assets/css/style.css
  4. Watch for changes: sass --watch assets/scss/style.scss:assets/css/style.css

Note: Editing style.css directly is perfectly fine if you are
not using Sass. The two files should not be edited simultaneously
or they will conflict.
